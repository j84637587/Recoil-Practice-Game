# Facebook-Screenshot
此專案利用pygame製作遊戲Apex Legends中，槍械開火後座力練習遊戲。

使用的後座力表為槍械在靶場紀錄10次的槍孔位置並平均後的後座力表

目前可選槍枝
- R-99
- R-301
- Flatline

後座力可於遊戲畫面左下角調整，且畫面左側會顯示預測彈道。

## 需求
- Python >= 3.9

## 使用方式

1. 安裝所需之套件
2. 執行指令開啟 ``` python3 main.py ```
